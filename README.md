# Proyectos Civic Tech

Proyectos digitales de participación ciudadana, inclusión digital y mejora de servicios públicos.

Este repositorio reúne ideas, prototipos y documentación para pequeños proyectos digitales con impacto social práctico.

## Objetivo

Crear herramientas simples que ayuden a identificar problemas, organizar información y mejorar la comunicación entre personas usuarias, entidades y servicios públicos.

## Líneas de trabajo

- Inclusión digital
- Accesibilidad móvil
- Mejora de servicios públicos
- Participación ciudadana
- Mapeo de incidencias urbanas
- Formularios simples para recopilar información
- Guías para comunicar problemas de forma clara

## Ideas de proyectos

- Mapa ciudadano de incidencias urbanas
- Formulario para reportar basura, vidrio roto o problemas peatonales
- Guía para comunicar incidencias al ayuntamiento
- Tabla de seguimiento de problemas locales
- Revisión de accesibilidad de servicios digitales
- Materiales de apoyo para personas migrantes o con baja competencia digital

## Principios

- Soluciones pequeñas y realistas
- Bajo coste
- Uso desde móvil
- Lenguaje claro
- Protección de datos personales
- Documentación abierta y reutilizable
