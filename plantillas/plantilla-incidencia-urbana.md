# Plantilla de incidencia urbana

Plantilla para registrar problemas urbanos de forma clara y no sensible.

## Tipo de incidencia

- Basura
- Vidrio roto
- Obstáculo peatonal
- Iluminación
- Seguridad peatonal
- Accesibilidad
- Otro

## Ubicación aproximada

Indicar zona general, no domicilio privado.

## Descripción

Describir el problema de forma objetiva.

## Impacto

- Riesgo para peatones
- Dificultad de accesibilidad
- Suciedad
- Riesgo para animales
- Otro

## Prioridad

- Alta
- Media
- Baja

## Evidencia

Foto o descripción, sin datos personales visibles.

## Próxima acción

- Registrar
- Comunicar a entidad responsable
- Hacer seguimiento
- Cerrar si se resuelve
