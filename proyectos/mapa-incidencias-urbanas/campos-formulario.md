# Campos del formulario de incidencias urbanas

## Campos recomendados

| Campo | Tipo | Obligatorio | Observaciones |
|---|---|---|---|
| Fecha | Fecha | Sí | Día de registro |
| Tipo de incidencia | Lista | Sí | Basura, vidrio, accesibilidad, etc. |
| Zona aproximada | Texto | Sí | Sin domicilio privado |
| Descripción | Texto largo | Sí | Objetiva y breve |
| Prioridad | Lista | Sí | Alta, media, baja |
| Evidencia | Archivo/enlace | No | Sin datos personales |
| Estado | Lista | Sí | Nueva, revisada, comunicada, resuelta |
| Próxima acción | Texto | No | Qué hacer después |

## Campos que no deben pedirse

- Nombre completo de persona afectada
- DNI/NIE
- Teléfono personal
- Dirección exacta de domicilio
- Datos médicos
- Información sensible innecesaria
