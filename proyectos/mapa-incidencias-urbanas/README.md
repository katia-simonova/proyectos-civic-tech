# Mapa ciudadano de incidencias urbanas

Proyecto de portfolio civic tech para organizar incidencias urbanas de forma simple y respetando la privacidad.

## Objetivo

Crear una estructura básica para recopilar, clasificar y seguir problemas urbanos como basura, vidrio roto, obstáculos peatonales o problemas de accesibilidad.

## Componentes

- Formulario simple
- Tabla de seguimiento
- Clasificación por tipo de incidencia
- Prioridad
- Estado
- Próxima acción

## Estados posibles

- Nueva
- Revisada
- Comunicada
- En seguimiento
- Resuelta
- Cerrada

## Principios

- No publicar datos personales.
- No mostrar domicilios privados.
- No identificar a personas sin consentimiento.
- Usar lenguaje claro.
- Priorizar problemas con impacto práctico.

## Estado

Prototipo documental de portfolio.
